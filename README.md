# Sistema de Inventario Deportivo - Universidad FET

##  Descripción

Este proyecto consiste en un sistema web para la gestión del inventario de equipos deportivos de la oficina de bienestar universitario. Permite controlar préstamos, stock y administración de equipos.

---

##  Tecnologías utilizadas

- Python
- Flask
- HTML, CSS, JavaScript
- Base de datos (MySQL)

---

##  Funcionalidades actuales

- ✅ Sistema de login
- ✅ Conexión con base de datos mediante Flask
- ✅ Gestión de inventario (CRUD)
- ✅ Alerta de stock bajo
- ✅ Visualización de reportes

---

##  Funcionalidades en desarrollo

- ✖️ Descarga de reportes en PDF
- ✖️ Sistema de roles completo (admin / usuario)
- ✖️ Control avanzado de préstamos

---

##  Tipos de usuarios

### Administrador
- Gestiona inventario
- Controla stock
- Visualiza reportes

### Usuario
- Solicita préstamos
- Consulta disponibilidad
